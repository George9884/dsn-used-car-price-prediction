# Used Car Price Prediction

This project was completed as part of the Data Science Nigeria (DSN) 2026 Free AI Classes program.

## Project Overview
The goal of this project is to predict the prices of used cars based on various vehicle attributes using machine learning techniques.

## Tools and Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Google Colab

## Workflow
1. Data preprocessing
2. Missing value handling
3. Feature encoding
4. Model training using XGBoost
5. Model evaluation
6. Kaggle submission generation
